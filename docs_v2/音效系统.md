## 二、音效清单

### 2.1 8种核心音效

| 音效名 | 波形 | 频率 | 时长 | 触发时机 | 效果描述 |
|--------|------|------|------|---------|---------|
| **correct** | sine | 880→1760Hz | 0.2s | 判断正确 | 清脆上升的叮声，愉悦感 |
| **wrong** | sawtooth | 150→100Hz | 0.5s | 判断错误 | 低沉下降的嗡鸣，挫败感 |
| **alarm** | square | 440↔880Hz 交替 | 0.6s | 毒品错误 | 警笛式交替，紧迫感 |
| **click** | sine | 600→800Hz | 0.1s | UI按钮点击 | 短促清脆，确认感 |
| **levelup** | sine | 523→1047Hz | 0.4s | 关卡通过/隐藏规则发现 | 上升旋律，成就感 |
| **scan** | square | 1200→1800Hz | 0.2s | 扫描进行中 | 高频扫描声，科技感 |
| **echo** | sine | 660→1320Hz | 0.6s | 直觉触发 | 神秘上升，特殊感 |
| **cycle** | sine | C5→E5→G5→C6 | 0.6s | 周期结算 | 四音符和弦，仪式感 |

### 2.2 音效参数详解

**correct（判断正确）**
```javascript
const osc = audioContext.createOscillator();
const gain = audioContext.createGain();

osc.type = 'sine';
osc.frequency.setValueAtTime(880, currentTime);           // A5
osc.frequency.exponentialRampToValueAtTime(1760, currentTime + 0.2);  // A6

gain.gain.setValueAtTime(0.3, currentTime);
gain.gain.exponentialRampToValueAtTime(0.01, currentTime + 0.2);

osc.connect(gain);
gain.connect(audioContext.destination);
osc.start();
osc.stop(currentTime + 0.2);
```

**wrong（判断错误）**
```javascript
const osc = audioContext.createOscillator();
const gain = audioContext.createGain();

osc.type = 'sawtooth';  // 锯齿波，更刺耳
osc.frequency.setValueAtTime(150, currentTime);
osc.frequency.exponentialRampToValueAtTime(100, currentTime + 0.5);

gain.gain.setValueAtTime(0.3, currentTime);
gain.gain.exponentialRampToValueAtTime(0.01, currentTime + 0.5);

osc.connect(gain);
gain.connect(audioContext.destination);
osc.start();
osc.stop(currentTime + 0.5);
```

**alarm（毒品错误）**
```javascript
const osc = audioContext.createOscillator();
const gain = audioContext.createGain();

osc.type = 'square';  // 方波，类似警笛

// 440Hz ↔ 880Hz 交替3次
for (let i = 0; i < 3; i++) {
  osc.frequency.setValueAtTime(440, currentTime + i * 0.2);
  osc.frequency.setValueAtTime(880, currentTime + i * 0.2 + 0.1);
}

gain.gain.setValueAtTime(0.3, currentTime);
gain.gain.linearRampToValueAtTime(0, currentTime + 0.6);

osc.connect(gain);
gain.connect(audioContext.destination);
osc.start();
osc.stop(currentTime + 0.6);
```

**cycle（周期结算）**
```javascript
const notes = [523, 659, 784, 1047];  // C5, E5, G5, C6
const duration = 0.6;

notes.forEach((freq, i) => {
  const osc = audioContext.createOscillator();
  const gain = audioContext.createGain();

  osc.type = 'sine';
  osc.frequency.setValueAtTime(freq, currentTime + i * 0.15);

  gain.gain.setValueAtTime(0, currentTime + i * 0.15);
  gain.gain.linearRampToValueAtTime(0.3, currentTime + i * 0.15 + 0.05);
  gain.gain.linearRampToValueAtTime(0, currentTime + i * 0.15 + 0.15);

  osc.connect(gain);
  gain.connect(audioContext.destination);
  osc.start();
  osc.stop(currentTime + i * 0.15 + 0.15);
});
```

## 三、SoundManager 架构

```javascript
class SoundManager {
  constructor() {
    this.ctx = null;  // AudioContext，延迟初始化
    this.enabled = true;
  }

  // 解锁音频（用户首次交互后调用）
  unlock() {
    if (!this.ctx) {
      this.ctx = new (window.AudioContext || window.webkitAudioContext)();
    }
    if (this.ctx.state === 'suspended') {
      this.ctx.resume();
    }
  }

  // 播放音效
  play(name) {
    if (!this.enabled || !this.ctx) return;

    const currentTime = this.ctx.currentTime;

    switch (name) {
      case 'correct': this._playCorrect(currentTime); break;
      case 'wrong': this._playWrong(currentTime); break;
      case 'alarm': this._playAlarm(currentTime); break;
      case 'click': this._playClick(currentTime); break;
      case 'levelup': this._playLevelup(currentTime); break;
      case 'scan': this._playScan(currentTime); break;
      case 'echo': this._playEcho(currentTime); break;
      case 'cycle': this._playCycle(currentTime); break;
    }
  }

  // 设置开关
  setEnabled(enabled) {
    this.enabled = enabled;
  }

  // 各音效实现方法...
}

// 全局单例
const soundManager = new SoundManager();
```

## 四、音频解锁策略

### 4.1 浏览器限制

现代浏览器要求**用户手势**后才能播放音频：
- Chrome：需要用户点击/触摸页面
- Safari：需要用户点击/触摸页面
- Firefox：相对宽松，但仍建议主动解锁

### 4.2 解锁实现

```javascript
// init.js 中
function initAudio() {
  const unlockHandler = () => {
    soundManager.unlock();
    document.removeEventListener('click', unlockHandler);
    document.removeEventListener('touchstart', unlockHandler);
  };

  document.addEventListener('click', unlockHandler);
  document.addEventListener('touchstart', unlockHandler);
}
```

### 4.3 降级处理

```javascript
play(name) {
  try {
    if (!this.enabled || !this.ctx) return;
    // ...播放逻辑
  } catch (e) {
    // 音频失败不阻断游戏流程
    console.warn('音效播放失败:', e);
  }
}
```
