## 一、设置入口

### 1.1 入口位置

| 位置 | 触发方式 | 使用场景 |
|------|---------|---------|
| **主界面右上角** | 点击 ⚙️ 图标 | 最常用入口，全局可达 |
| **游戏暂停界面** | 点击"设置"按钮 | 局内快速调整音效/操作 |
| **关卡结算界面** | 点击"设置"按钮 | 结算后调整 |

### 1.2 设置按钮设计

```
┌─────┐
│ ⚙️  │  ← 齿轮图标，40×40px(PC) / 48×48px(移动端)
│     │     默认透明度80%，hover时100%
└─────┘
```

**交互效果**：
- **Hover**：图标顺时针旋转45°，0.3s过渡
- **Click**：图标旋转360°+缩放1.1倍，同时弹出设置面板
- **新功能提示**：图标右上角红点闪烁（`animation: pulse 1.5s infinite`）

```css
.settings-btn {
  width: 40px;
  height: 40px;
  opacity: 0.8;
  transition: all 0.3s ease;
}
.settings-btn:hover {
  opacity: 1;
  transform: rotate(45deg);
}
.settings-btn:active {
  transform: rotate(360deg) scale(1.1);
  transition: all 0.5s ease;
}
.settings-btn .badge {
  position: absolute;
  top: -4px;
  right: -4px;
  width: 10px;
  height: 10px;
  background: #ff4444;
  border-radius: 50%;
```

### 2.2 导航栏设计

| 图标 | 名称 | 说明 | 未读标记 |

| 🔊 | 音效设置 | 游戏音效、提示音开关 | — |
| 🎵 | 音乐设置 | BGM音量、曲目选择 | 新曲目解锁时红点 |
| 🎮 | 操作设置 | 操作方式、灵敏度 | — |

| 💾 | 存档管理 | 存档、读档、重置 | — |

| ℹ️ | 关于我们 | 版本信息、制作团队 | — |

**导航交互**：
- 当前选中项：左侧蓝色竖条指示 + 文字高亮
- 切换动画：内容区淡入淡出，0.2s过渡
- 未读标记：新功能/新成就/新曲目解锁时显示红点

### 2.3 打开/关闭动画

```
【打开】
1. 背景遮罩淡入（0→50%黑色，0.2s）
2. 设置面板从右侧滑入（translateX(100%)→0，0.3s，ease-out）
3. 左侧导航栏依次淡入（stagger 0.05s）

【关闭】
1. 设置面板向右滑出（0→translateX(100%)，0.2s，ease-in）
2. 背景遮罩淡出（50%→0，0.2s）
3. 点击遮罩/关闭按钮/X键均可触发关闭
```

```javascript
    master_volume: 80,
    game_volume: 70,
    hint_volume: 50,
    ui_volume: 40,
    vibration: true,
    voice_announce: true
  }
```

## 四、设置即时生效规则

| 设置项 | 生效时机 | 需确认 |
|--------|---------|--------|
| 音量调节 | 拖动时即时 | 否 |
| 画质切换 | 选择后即时 | 否 |
| 操作模式 | 选择后即时 | 否（下局生效） |
| 语言切换 | 选择后即时 | 是（弹窗确认） |
| 存档操作 | 点击后执行 | 是（二次确认） |
