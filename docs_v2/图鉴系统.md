# 📚 缉私十二时辰 · 图鉴系统文档

> 收集系统完整设计 — 物品图鉴 / 剧情碎片 / 导师档案 / 直觉图鉴

---

## 一、系统定位

图鉴系统是游戏的**收集与成就核心**，承担三大功能：
1. **知识沉淀**：将游戏过程中接触的反走私知识系统化归档
2. **情感连接**：通过剧情碎片拼凑完整大案，驱动玩家持续游玩
3. **成就展示**：直观展示游戏进度，增强玩家成就感

---

## 二、图鉴分类

### 2.1 四大分类

| 分类 | 内容 | 解锁方式 | 展示形式 |
|------|------|---------|---------|
| **物品图鉴** | 所有遇到过的走私/合法物品 | 关卡中自然遇见 | 图标网格+详情卡片 |
| **剧情碎片** | 36段叙事，拼成完整反走私大案 | 首次通关对应关卡 | 碎片墙+阅读模式 |
| **导师档案** | 6位导师的详细资料与故事 | 首次进入对应导师关卡 | 人物卡+关系度 |
| **直觉图鉴** | 已解锁的6种直觉效果说明 | 商店购买或周期奖励 | 能力卡片+触发记录 |

```javascript
// 图鉴相关存档数据
{
  seenItems: ['phone_legal', 'cigarette_illegal', ...],  // 已见物品ID
  fragments: ['k01', 'k02', 'k07', ...],                  // 已收集碎片ID
  archive: ['phone_legal', 'ivory_illegal', ...],         // 已解锁图鉴条目
  quizResults: {                                           // 测验成绩
    k01: 5,  // 3/3正确 + 首次奖励
    k02: 2   // 2/3正确
  },
  tutorAffinity: {                                         // 导师关系度
    linrui: 45,    // 0-100
    chenfeng: 30,
    zhaohai: 20,
    baiwei: 15,
    laozhou: 10,
    xiaohui: 5
  }
}
```

### 8.2 解锁触发点

| 内容 | 触发条件 | 存档写入 |
|------|---------|---------|
| 物品遇见 | 关卡中首次显示该物品 | seenItems.push(id) |
| 碎片收集 | 关卡首次通关 | fragments.push(levelId) |
| 图鉴解锁 | seenItems中新增物品 | archive自动同步 |
| 导师关系 | 通关/购买知识点/测验 | tutorAffinity[tutor] += value |
