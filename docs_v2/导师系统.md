## 一、设计原则

### 1.1 为什么原创？

| 风险 | 说明 |
|------|------|
| **肖像权** | 使用真实姓名和形象需授权 |
| **名誉权** | 游戏内角色行为可能被关联到真实人物 |
| **比赛合规** | 部分比赛要求作品完全原创 |
| **改编自由度** | 虚构角色可以加入更多戏剧元素 |

### 1.2 借鉴 vs 原创

- ❌ 直接使用："郭惠绮，深圳罗湖海关关员"
- ✅ 借鉴特质："一位50+的女关员，眼神锐利，年查私额千万级"
- ✅ 融合多原型：把3位英雄的特点融合成1个角色
- ✅ 艺术夸张：基于真实事迹但加入虚构情节

**负责关卡**：k01~k06

## 六、情感曲线设计

```
第5关（综合）：挑战 → 老周分享守岛精神 → 坚守
第6关（终极）：高潮 → 小辉纪念 → 情感升华
```
