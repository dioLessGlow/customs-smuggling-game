# 🔄 缉私十二时辰 · 周期系统文档

> 10关挑战周期 — 存活机制 / 结算奖励 / 失败处理

---

## 一、核心概念

| 概念 | 说明 |
|------|------|
| **周期 (Cycle)** | 10关为一个完整周期 |
| **轮回** | 周期失败后的重置机制 |
| **周期功勋 (cycleDao)** | 周期内累积的功勋，失败时清零 |
| **永久功勋 (permanentDao)** | 跨周期保留，可消费 |
| **存活条件** | 周期内通关 ≥ 6关（60%通过率） |

## 二、周期规则

### 2.1 基本参数

```javascript
const CYCLE_SIZE = 10;        // 每周期10关
const CYCLE_PASS_THRESHOLD = 6; // 通过≥6关算成功
```

### 2.2 周期生命周期

```
周期开始（currentCycle = N）
    │
    ├─ roundsInCycle = []     // 清空本周期记录
    ├─ cycleDao = 0           // 清空周期功勋
    │
    ├─ 第1关 → 选择关卡 → 游玩 → 通过/失败
    │   └─ addRound(levelId, passed, score, dao)
    │       └─ push到roundsInCycle
    │
    ├─ 第2关 → ...
    ├─ 第3关 → ...
    ├─ ...（最多10关）
    │
    └─ 达到10关 或 玩家主动结束
        └─ endCycle()
            ├─ 统计passedCount
            ├─ passedCount ≥ 6 → 周期成功
            │   ├─ currentCycle++
            │   ├─ 随机解锁1个新直觉
            │   ├─ cycleDao清零
            │   └─ 显示成功结算
            │
            └─ passedCount < 6 → 周期失败
                ├─ cycleDao清零（失去的功勋显示）
                ├─ roundsInCycle清空
                └─ 显示失败结算
```

## 四、周期数据流

### 4.1 CycleManager 接口

```javascript
class CycleManager {
  // 添加一轮记录
  static addRound(save, levelId, passed, score, dao) {
    save.roundsInCycle.push({
      levelId,
      passed,
      score,
      dao,
      timestamp: Date.now()
    });

    // 检查是否满10关
    if (save.roundsInCycle.length >= CYCLE_SIZE) {
      return this.endCycle(save);
    }
    return { ended: false };
  }

  // 结束周期
  static endCycle(save) {
    const passedCount = save.roundsInCycle.filter(r => r.passed).length;
    const totalDao = save.roundsInCycle.reduce((sum, r) => sum + r.dao, 0);

    if (passedCount >= CYCLE_PASS_THRESHOLD) {
      // 成功
      save.currentCycle++;
      save.permanentDao += save.cycleDao; // 周期功勋转为永久
      const newEcho = this._unlockRandomEcho(save);
      save.cycleDao = 0;
      save.roundsInCycle = [];

      return {
        success: true,
        cycle: save.currentCycle - 1,
        passed: passedCount,
        total: CYCLE_SIZE,
        newEcho,
        convertedDao: totalDao
      };
    } else {
      // 失败
      const lostDao = save.cycleDao;
      save.cycleDao = 0;
      save.roundsInCycle = [];

      return {
        success: false,
        cycle: save.currentCycle,
        passed: passedCount,
        total: CYCLE_SIZE,
        lostDao
      };
    }
  }

  // 随机解锁直觉
  static _unlockRandomEcho(save) {
    const available = ECHO_POOL.filter(e => !save.echoSkills.includes(e.id));
    if (available.length === 0) return null;

    const random = available[Math.floor(Math.random() * available.length)];
    save.echoSkills.push(random.id);
    return random;
  }
}
```

## 五、周期与商店的关系

| 场景 | 周期功勋 | 永久功勋 |
|------|---------|---------|
| 单局结算 | +N | — |
| 周期成功 | 全部转为永久 | +cycleDao |
| 周期失败 | 清零 | 不变 |
| 商店消费 | — | 可消费 |

**策略提示**：
- 周期功勋是"赌注"，周期成功才能变现
- 永久功勋是"存款"，随时可消费
- 风险与收益：周期内玩得越多，cycleDao越多，但失败风险也越大
