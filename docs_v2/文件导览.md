## 管理器层 — `src/managers/`

### `SaveManager.js` (120 行)

**全局单例 `SaveManager`**

- `load()` / `save()` / `getDefault()` — 序列化持久化
- `_migrate()` — 增量版本迁移（含 `tutorNames` 字段补丁）
- `unlockNextLevel()` / `addAffinity()` — 游戏进度

**默认存档结构**:
```js
{
  unlocked: ['k01','k07','k13','k19','k25','k31'],
  completed: [], totalScore: 0, permanentDao: 3,
  currentCycle: 1, shopItems: [], shopKnowledge: [],
  tutorAffinity: {}, tutorNames: {},
  settings: { music_volume:70, music_enabled:true, music_track:0, ... },
  seenItems: []
}
```

### `CycleManager.js` (47 行)

- `CYCLE_SIZE = 10` — 10 关一轮
- 阈值 60% (≥6 关通过) → 成功获得新直觉，失败重置 cycleDao
- `endCycle()` → 检查 + 奖励发放 + `_unlockRandomEcho()`

### `EchoManager.js` (33 行)

- 兼容接口，实际由商店系统接管
- `getActiveEcho()` / `pickRandomEcho()` / `clearEcho()`

### `SoundManager.js` (226 行)

**实例 `soundManager`** — 纯 Web Audio API 合成

| 方法 | 说明 |
|------|------|
| `play(type)` | 8 种 SFX: correct/wrong/alarm/click/levelup/scan/echo/cycle |
| `playTrack(index)` | 5 首程序化 BGM |
| `stopBGM()` | 停止当前 BGM |
| `setBGMVolume(v)` | 实时调整音量 |
| `setMuted(bool)` | 全局静音 |

**5 首 BGM**:
| Index | 名称 | 风格 |
|-------|------|------|
| 0 | 缉私进行曲 | 方波进行曲, 120bpm |
| 1 | 十二时辰·晨 | 竖琴琶音, 100bpm |
| 2 | 十二时辰·午 | 锯齿波律动, 130bpm |
| 3 | 十二时辰·暮 | 泛音钟声, 90bpm |
| 4 | 十二时辰·夜 | 星空旋律, 70bpm |

AudioContext 延迟创建于首次用户交互 (click/touch)。

| 功能 | 细节 |

| 调色板 | 9 帧昼夜渐变 (晨曦→白天→暮色→深夜) |

## 数据流

```
用户操作 → UI.show(name)
         → 对应 handler 渲染 DOM
         → SaveManager.load() 读取存档
         → 交互修改存档 → SaveManager.save()
         → Phaser 场景 (GenericLevelScene) 独立运行
         → 事件 'game:levelEnd' → result 屏幕
         → 事件 'game:cycleEnd' → cycle-summary 屏幕
```
