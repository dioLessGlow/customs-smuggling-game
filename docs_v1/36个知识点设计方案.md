## 一、林锐 · 通关走私识别（6个知识点）

### 知识点1：伪报品名

```
【走私手法】
将高税率/限制类商品伪报为低税率/非限制类商品

【典型案例】
• "稀土"伪报为"普通金属矿"
• "象牙"伪报为"工艺品"
• "冻牛肉"伪报为"冷冻食品"
• "手机"伪报为"手机配件"

【识别要点】
✓ 品名描述模糊或过于笼统
✓ 申报品名与实际货物外观不符
✓ 品名对应的税率明显偏低

【反制手段】
✓ HS编码数据库比对
✓ 开箱查验确认实际品名
✓ 价格数据库交叉验证

【法律依据】
《海关法》第24条：进口货物的收货人应当向海关如实申报

【数据】
伪报品名占通关走私案件的35%
```

**学习效果：** 伪报关卡的识别准确率+20%

---

### 知识点2：伪报价格

```
【走私手法】
故意低报进口价格，偷逃应缴税额

【典型案例】
• 实际价格100万，申报30万
• 利用关联交易转移定价
• 伪造虚假发票

【识别要点】
✓ 申报价格明显低于市场价
✓ 同类型商品历史价格对比异常
✓ 买卖双方为关联企业

【反制手段】
✓ 海关估价系统（参考国际价格）
✓ 要求提供付汇凭证、合同
✓ 与税务、银行数据交叉比对

【法律依据】
《海关审定进出口货物完税价格办法》

【数据】
低报价格平均偷逃税额达真实价格的60%
```

**学习效果：** 价格异常关卡的识别准确率+20%

---

### 知识点3：伪报数量

```
【走私手法】
少报多进或多报少出

【典型案例】
• 实际进口1000件，申报500件
• 出口时多报数量骗取退税
• 利用"分批进口"分散申报

【识别要点】
✓ 货物重量与申报数量不匹配
✓ 包装规格与申报数量矛盾
✓ 集装箱容积利用率异常

【反制手段】
✓ 称重查验
✓ 清点查验
✓ 集装箱容积分析

【法律依据】
《海关法》第86条：申报不实可处以罚款

【数据】
伪报数量案件平均少报率达40%
```

**学习效果：** 数量异常关卡的识别准确率+20%

### 知识点4：瞒报夹藏

```

✓ X光扫描显示异常阴影
✓ 缉私犬对特定区域有反应
✓ 包装异常（过度或简陋）

【反制手段】
✓ H986大型X光集装箱检查系统
✓ 缉私犬查验
✓ 随机开箱查验
✓ 舱单数据比对

【法律依据】
《海关法》第82条：走私行为追究刑事责任

【数据】
瞒报夹藏占集装箱走私案件的60%
```

**学习效果：** 夹藏关卡的识别准确率+20%
