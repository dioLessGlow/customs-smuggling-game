## 一、设置入口

### 1.1 入口位置

| 位置 | 触发方式 | 说明 |

## 五、设置面板交互逻辑

### 5.1 打开/关闭动画

```
打开：
1. 背景遮罩淡入（0→50%黑色，0.2s）
2. 设置面板从右侧滑入（translateX(100%)→0，0.3s，ease-out）
3. 左侧导航栏依次淡入（stagger 0.05s）

关闭：
1. 设置面板向右滑出（0→translateX(100%)，0.2s，ease-in）
2. 背景遮罩淡出（50%→0，0.2s）
3. 点击遮罩/关闭按钮/X键均可触发关闭
```

### 5.2 设置项即时生效

| 设置项 | 生效时机 | 是否需要确认 |
|--------|---------|------------|
| 音量调节 | 拖动时即时 | 否 |
| 画质切换 | 选择后即时 | 否 |
| 操作模式 | 选择后即时 | 否（下局生效） |
| 语言切换 | 选择后即时 | 是（弹窗确认） |
| 存档操作 | 点击后执行 | 是（二次确认） |
| 数据清空 | 点击后执行 | 是（三次确认） |

### 5.3 设置自动保存

- 所有设置项修改后自动保存到本地存储
- 无需手动点击"保存"按钮
- 异常退出时设置不丢失
