# 岐青乡望 · 缉私终焉
## 第四届"思创杯"珠澳大学生反走私创意大赛 — H5游戏参赛作品

> **项目代号**：缉私终焉  
> **技术栈**：Phaser 3 + 原生 JavaScript + GitHub Pages  
> **目标平台**：手机竖屏（主）/ 电脑浏览器（兼容）  
> **开发周期**：个人参赛，2-3周迭代

---

## 一、核心创意来源

### 1.1 灵感来源：《十日终焉》
番茄小说现象级作品《十日终焉》的核心吸引力：

| 小说看点 | 游戏化映射 |
|---------|-----------|
| **规则扭曲** — 表面规则下隐藏反直觉设定 | 每关有"隐藏规则"，通关后揭示"原来如此" |
| **十日轮回** — 死亡重置但保留记忆 | 10关为一个周期，失败重置但保留已解锁内容 |
| **群像博弈** — 多角色信息不对等对抗 | 加入AI对手，比识别速度/准确率 |
| **回响觉醒** — 极端情绪下触发随机技能 | 随机技能触发（如"直觉"：首次错误不扣分） |
| **世界观悬念** — 底层谜题驱动追更 | 每通关解锁"剧情碎片"，拼凑反走私大案 |

### 1.2 主题契合：珠澳口岸反走私
大赛重点打击方向 → 游戏关卡设计：

| 走私类型 | 小说生肖映射 | 关卡名称 |
|---------|------------|---------|
| 水客夹带（藏匿/伪装） | 人鼠 | 鼠窃 |
| 冻品走私（未经检疫） | 人牛 | 牛冻 |
| 象牙制品（濒危物种） | 人虎 | 虎牙 |
| 异宠走私（非法物种） | 人兔 | 兔穴 |
| 枪支弹药 | 人龙 | 龙火 |
| 毒品走私（人体藏毒） | 人蛇 | 蛇毒 |
| 重点商品（烟草/高档消费品） | 人羊 | 羊裘 |
| 电子产品（伪报/瞒报） | 人猴 | 猴机 |
| 禁止进出境物品（文物/反宣品） | 人鸡 | 鸡笼 |
| 涉海走私 | 人狗 | 犬嗅 |
| 洋垃圾走私 | 人猪 | 猪宴 |

### 2.1 三级难度体系（人/地/天）

```

├─────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  │
│  │  人级   │  │  地级   │  │  天级   │  │

│  │ 12生肖   │  │ 12生肖   │  │ 12生肖   │  │
│  └─────────┘  └─────────┘  └─────────┘  │
└─────────────────────────────────────────┘
```

| 级别 | 解锁条件 | 机制复杂度 | 代表机制 |
|------|---------|-----------|---------|
| **人级** | 初始解锁 | 单一机制，3秒判断 | 纯图标识别 |
| **地级** | 人级通关 | 双重机制，需要策略 | X光 + 重量双重核对 |
| **天级** | 地级通关 | 三重机制 + 随机事件 | 扫描 + 时间压力 + 假警报干扰 |

### 2.2 核心循环（30秒上手）

         ↑_________________________________________________↓

## 三、关卡设计（MVP：人级3关）

### 3.1 人鼠 · 鼠窃（水客夹带）

**教学目的**：基础操作教学 — 放行/扣押

**表面规则**：
- 行李传送带，3秒/件
- 判断：合法物品 → 放行（✅）/ 走私物品 → 扣押（🚫）
- 5件中答对4件过关

**隐藏规则（通关后揭示）**：
> "走私者会故意在前2件放明显仿品，消耗你的注意力。第3件往往是真货。"

**UI布局**：
```
┌─────────────────────────┐
│  关卡：人鼠 · 鼠窃       │
│  ⏱️ 60秒  💰 得分：0     │
├─────────────────────────┤
│                         │
│    ┌─────────────┐      │
│    │  📦 行李扫描 │      │  ← 物品显示区
│    │  [3D旋转动画]│      │
│    └─────────────┘      │
│                         │
│    普通手机 — 合法？      │  ← 物品名称
│                         │
├─────────────────────────┤
│  ┌────────┐ ┌────────┐  │
│  │  ✅ 放行 │ │ 🚫 扣押 │  ← 操作按钮
│  └────────┘ └────────┘  │
└─────────────────────────┘
```

**代码结构**：
```javascript
class RatScene extends BaseLevel {
    constructor() {
        super({ key: 'RatScene' });
        this.levelId = 'human_rat';
        this.timeLimit = 60;
        this.itemsPerWave = 5;
        this.requiredCorrect = 4;
    }

    spawnItem() {
        // 前2件：明显仿品（高识别度）
        // 第3件：真走私品（需要仔细观察）
        // 第4-5件：混合
        const wave = this.currentWave;
        const item = this.getItemByWavePattern(wave);
        this.showItem(item);
    }

    judge(choice) {
        const result = super.judge(choice);
        if (this.itemsJudged === 3 && result.correct) {
            this.triggerHiddenRuleReveal(); // 触发隐藏规则揭示
        }
    }
}
```

### 3.2 人虎 · 虎牙（象牙制品）

**教学目的**：引入工具 — 主动使用放大镜

**表面规则**：
- 物品外观相似，必须点击"放大镜"查看纹理
- 象牙有独特交叉纹理（施雷格线），塑料没有

**隐藏规则**：
> "放大镜使用次数有限（3次/关）。走私者会在第5件放'免检VIP'行李，强制放行，浪费你的放大镜。"

**核心机制**：
```
不放大时：🐘 象牙 和 🧸 塑料玩具 显示为相同图标
放大后：  🐘 → 显示交叉纹理 / 🧸 → 显示均匀纹理
```

**UI变化**：
```
┌─────────────────────────┐
│  🔍 放大镜：3/3 剩余     │  ← 工具状态
├─────────────────────────┤
│    ┌─────────────┐      │
│    │    🐘？      │      │  ← 未放大：显示"？"
│    │   [点击放大]  │      │
│    └─────────────┘      │
│                         │
│  放大后：显示纹理细节      │
│  ┌─────────────────┐   │
│  │ ╱╲╱╲╱ 交叉纹理  │   │  ← 象牙特征
│  │ ╲╱╲╱╲           │   │
│  └─────────────────┘   │
└─────────────────────────┘
```

### 3.3 人蛇 · 蛇毒（毒品走私）

**教学目的**：引入风险 — 错误代价差异化

**表面规则**：
- 人体扫描轮廓，识别异常凸起
- 正常部位 vs 藏毒部位

**隐藏规则**：
> "毒品走私的惩罚是普通错误的10倍。但如果你能连续正确识别3件，会触发'缉私直觉'，下次错误不扣分。"

**风险机制**：
| 错误类型 | 扣分 | 特效 |
|---------|------|------|
| 普通走私品错误 | -5 | 红色闪烁 |
| 毒品走私错误 | -50 | 屏幕震动 + 警报声 |
| 连续3次正确后触发"直觉" | 下次错误不扣分 | 金色光环 |

## 六、音效设计（程序生成）

使用 Web Audio API 合成音效，无需外部文件：

```javascript
class SoundManager {
    constructor() {
        this.ctx = new (window.AudioContext || window.webkitAudioContext)();
    }

    // 正确音效：清脆叮声
    playCorrect() {
        const osc = this.ctx.createOscillator();
        const gain = this.ctx.createGain();
        osc.connect(gain);
        gain.connect(this.ctx.destination);
        osc.frequency.setValueAtTime(880, this.ctx.currentTime); // A5
        osc.frequency.exponentialRampToValueAtTime(1760, this.ctx.currentTime + 0.1);
        gain.gain.setValueAtTime(0.3, this.ctx.currentTime);
        gain.gain.exponentialRampToValueAtTime(0.01, this.ctx.currentTime + 0.3);
        osc.start();
        osc.stop(this.ctx.currentTime + 0.3);
    }

    // 错误音效：低沉嗡鸣
    playWrong() {
        const osc = this.ctx.createOscillator();
        const gain = this.ctx.createGain();
        osc.type = 'sawtooth';
        osc.connect(gain);
        gain.connect(this.ctx.destination);
        osc.frequency.setValueAtTime(150, this.ctx.currentTime);
        gain.gain.setValueAtTime(0.3, this.ctx.currentTime);
        gain.gain.exponentialRampToValueAtTime(0.01, this.ctx.currentTime + 0.5);
        osc.start();
        osc.stop(this.ctx.currentTime + 0.5);
    }

    // 警报音效：毒品错误
    playAlarm() {
        const osc = this.ctx.createOscillator();
        const gain = this.ctx.createGain();
        osc.type = 'square';
        osc.connect(gain);
        gain.connect(this.ctx.destination);
        // 警笛频率波动
        osc.frequency.setValueAtTime(440, this.ctx.currentTime);
        osc.frequency.linearRampToValueAtTime(880, this.ctx.currentTime + 0.2);
        osc.frequency.linearRampToValueAtTime(440, this.ctx.currentTime + 0.4);
        gain.gain.setValueAtTime(0.3, this.ctx.currentTime);
        gain.gain.linearRampToValueAtTime(0, this.ctx.currentTime + 0.6);
        osc.start();
        osc.stop(this.ctx.currentTime + 0.6);
    }
}
```
