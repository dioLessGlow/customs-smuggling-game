# 《缉私终焉》使用《十日终焉》设定的法律风险分析

> 基于中国《著作权法》《反不正当竞争法》及司法判例分析

---

## 一、核心结论

**有风险，但可以规避。关键在于"去特定化"。**

| 使用方式 | 风险等级 | 说明 |
|---------|---------|------|
| 直接使用"十日终焉"名称 | 🔴 高风险 | 涉嫌侵犯商标权/不正当竞争 |
| 使用"生肖""轮回"等通用概念 | 🟢 低风险 | 属于公有领域概念 |
| 使用"回响""道""终焉"等特定设定 | 🟡 中风险 | 需判断是否构成独创性表达 |
| 使用具体角色名/剧情 | 🔴 高风险 | 明确侵权 |

---

## 二、法律依据

### 2.1 《著作权法》第十三条
> "改编、翻译、注释、整理已有作品而产生的作品，其著作权由改编、翻译、注释、整理人享有，但行使著作权时不得侵犯原作品的著作权。"

### 2.2 "思想与表达二分法"
> 著作权法只保护**表达**，不保护**思想**。 citeweb_search:12#3web_search:12#6

| 受保护（表达） | 不受保护（思想） |
|--------------|----------------|
| 具体角色描写、情节编排 | 角色名称、抽象性格 |
| 完整的故事线、场景设置 | "轮回""生肖"等通用概念 |
| 特定的世界观细节 | "升级""技能树"等游戏机制 |

### 2.3 司法判例参考

**金庸诉江南《此间的少年》案（2016）** citeweb_search:12#2web_search:12#15
- **判决结果**：不构成著作权侵权（仅使用人物名称），但构成不正当竞争
- **关键理由**：人物名称属于抽象思想，不具独创性；但借用知名度"搭便车"构成不正当竞争

**长沙法院2024年同人小说案** citeweb_search:12#13
- **判决结果**：同人小说侵犯改编权，平台承担帮助侵权责任
- **赔偿**：2万元

## 三、当前设计的风险点

### 🔴 高风险（必须修改）

| 当前使用 | 风险 | 建议修改 |
|---------|------|---------|
| 游戏名称含"终焉" | 可能让人联想到《十日终焉》 | 改为"缉私轮回""缉私十二时辰"等 |
| "十日轮回"机制 | 与原著核心设定高度相似 | 改为"十二时辰轮回"或"缉私周期" |
| "回响"技能名称 | 原著独创性设定 | 改为"缉私直觉""危机预感"等 |
| "道"作为货币 | 原著特定概念 | 改为"功勋""绩点""正义值"等 |
| "生肖"对应走私类型 | 虽通用但组合方式相似 | 改为"十二地支"或直接用走私类型名称 |

### 🟡 中风险（建议修改）

| 当前使用 | 风险 | 建议修改 |
|---------|------|---------|
| "信念"局内资源 | 与原著精神力量设定相似 | 改为"警觉""专注力"等 |
| "湮灭"失败状态 | 原著特定术语 | 改为"任务失败""周期重置" |
| "觉醒"机制 | 通用但组合后相似 | 改为"触发""激活" |

### 🟢 低风险（可保留）

| 当前使用 | 说明 |
|---------|------|
| 轮回/周期机制 | 属于通用游戏设计 |
| 升级系统 | 属于通用游戏机制 |
| 技能树 | 属于通用游戏机制 |
| 生肖作为分类 | 属于中国传统文化公有领域 |

## 四、修改方案

### 方案A：完全去《十日终焉》化（推荐）

**新设定框架：**

```
原设定 → 新设定

"十日终焉" → "缉私十二时辰"（或"缉私周期"）
"十日轮回" → "十二关挑战周期"
"道" → "功勋"（或"正义值""缉私绩点"）
"信念" → "警觉"（或"专注力"）
"回响" → "缉私直觉"（或"危机预感""专业判断"）
"终焉·湮灭" → "任务失败·周期重置"
"生肖" → 直接用走私类型名称（水客、冻品、毒品等）
"觉醒回响" → "触发直觉"
```

**保留的核心机制（通用，不侵权）：**
- 周期挑战机制（通用游戏设计）
- 局内临时资源 + 局外永久货币（通用）
- 技能升级系统（通用）
- 失败保留部分进度（通用）

### 方案B：保留致敬但明确区分

如果坚持保留部分致敬元素：

1. **游戏名称**必须修改，不能含"终焉"
2. **宣传时**明确说明"玩法灵感来源于各类轮回题材作品"
3. **不直接使用**"回响""道"等特定术语
4. **增加原创元素**占比超过50%
5. **添加免责声明**："本游戏为原创反走私宣传作品，与任何小说作品无关"

**但此方案仍有风险，不建议用于比赛作品。**

| **走私终结者** | 直白有力 |
| **边境缉私员** | 职业视角 |

|-------|--------|------|

### 新技能系统

| 原"回响" | 新"缉私直觉" | 触发条件 | 效果 |
|---------|-------------|---------|------|
| 灵闻 | **敏锐嗅觉** | 警觉满20 | 提前显示物品类别 |

### 新周期机制

```
原：十日轮回 → 新：十二关挑战周期

第1周期：

```

## 六、比赛作品特别建议

### 为什么比赛作品更要谨慎？

1. **评审可见性高**：比赛作品会被公开展示，版权方更容易发现
2. **学校背书**：作品代表学校，侵权会影响学校声誉
3. **潜在商业价值**：获奖作品可能被推广，商业使用风险更高
4. **评审评分**：如果评审认为有侵权嫌疑，可能直接扣分或取消资格

### 建议做法

```
1. 完全移除所有《十日终焉》特定术语
2. 游戏名称、宣传文案中不出现"终焉""回响""道"等词
3. 机制说明使用通用游戏术语（周期、升级、技能等）
4. 如果评审问灵感来源，回答"借鉴了各类轮回题材作品的机制设计"
5. 在作品说明中强调"原创反走私宣传游戏"
```

## 七、一句话总结

> **"十日轮回"是思想（通用机制，可用），"十日终焉"是表达（特定作品，不可用）。把"终焉"改成"时辰"，把"回响"改成"直觉"，把"道"改成"功勋"——机制不变，风险归零。**
