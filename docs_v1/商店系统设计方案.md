# 🏪 缉私十二时辰 · 功勋商店系统设计方案 v3.2

> 游戏名称：缉私十二时辰（原缉私终焉）
> 设计目标：去《十日终焉》化，贴合缉私主题，增强反走私宣传效果

---

## 一、货币体系

| 货币 | 原名 | 新名 | 说明 | 重置规则 |
|-----|------|------|------|---------|
| 永久货币 | 道 | **功勋** | 局结束结算获得，用于商店购买 | **永久保留** |
| 局内货币 | 信念 | **警觉** | 局内正确判断获得，用于局内升级 | **每局重置** |

### 功勋获取方式

| 途径 | 功勋值 | 说明 |
|------|--------|------|
| 单局结算 | 查获数×2 + 连击数×1 | 基础奖励 |
| 首次通关某关 | +10 | 一次性奖励 |
| 首次分享成绩 | +5 | 一次性奖励 |
| 知识测验全对 | +5 | 每次测验 |
| 周期通关≥7关 | +20 | 额外奖励 |
| 连续登录7天 | +10 | 累计奖励 |

### 知识点学习流程

```
购买知识点（消耗功勋）
    ↓
解锁导师"私教课"界面
    ↓
导师对话开场 → 案例讲解 → 数据展示 → 法律依据
    ↓
课后小测验（3道选择题）
    ↓
    ├─ 全对 → +5功勋 + 解锁该关卡隐藏线索
    └─ 错1题 → +2功勋
    └─ 错2题及以上 → 无额外奖励，可重新学习
    ↓
永久效果激活（对应关卡属性加成）
```

### 知识点内容示例

#### K01 · 林锐 · 伪报品名

> **核心内容**：将高税率商品伪报为低税率商品，如"稀土"伪报为"普通金属矿"。
>
> **识别要点**：
> - 品名描述模糊、笼统
> - 申报税率明显偏低
> - 品名与实际货物外观不符
>
> **反制手段**：HS编码数据库比对、开箱查验、实验室检测
>
> **真实案例**：某企业将稀土伪报为普通金属矿出口，偷逃关税数百万元。
>
> **法律依据**：《海关法》第82条、《刑法》第153条

#### K08 · 陈锋 · 行李夹藏毒品

> **核心内容**：昆明裙子藏毒案——12条裙子彩色竖条内缝制毒品竖条，藏毒1843克。
>
> **识别要点**：
> - 物品重量异常（裙子过重）
> - 结构异常（彩色竖条内有硬物）
> - 缉毒犬有反应
>
> **反制手段**：缉毒犬嗅探 + X光扫描 + 人工仔细检查
>
> **法律依据**：《刑法》第347条（走私毒品罪，最高死刑）

#### K36 · 小辉 · 缉私精神

> **核心内容**：忠诚、勇敢、智慧、奉献。
>
> **精神传承**：从"望远镜"到"查船机器人"，变的是科技，不变的是精神。
>
> **导师寄语**："年轻的见习缉私员，希望你每次任务后，都能平安归来。"
>
> **解锁效果**：所有关卡初始警觉 +2（全局最强加成）

## 九、购买逻辑代码（伪代码）

```javascript
function purchaseItem(itemId) {
  const item = shopConfig[itemId];

  // 1. 检查是否已拥有
  if (player.unlocked_items.includes(itemId)) {
    showToast("已拥有该商品");
    return;
  }

  // 2. 检查前置条件
  if (item.prerequisite && !player.unlocked_items.includes(item.prerequisite)) {
    showToast(`需要先购买: ${item.prerequisite_name}`);
    return;
  }

  // 3. 检查功勋是否足够
  if (player.gongxun < item.cost) {
    showToast("功勋不足，继续缉私任务获取吧！");
    return;
  }

  // 4. 弹出确认框
  showConfirmDialog({
    title: "确认购买？",
    item: item,
    onConfirm: () => {
      // 5. 扣除功勋
      player.gongxun -= item.cost;

      // 6. 标记已拥有
      player.unlocked_items.push(itemId);

      // 7. 应用效果
      applyItemEffect(item);

      // 8. 播放成功动画
      playSuccessAnimation(item);

      // 9. 保存存档
      saveGame();
    }
  });
}

function purchaseKnowledge(knowledgeId) {
  const knowledge = knowledgeConfig[knowledgeId];

  if (player.unlocked_knowledge.includes(knowledgeId)) {
    showToast("已学习该知识点");
    return;
  }

  if (player.gongxun < knowledge.cost) {
    showToast("功勋不足");
    return;
  }

  showConfirmDialog({
    title: "解锁私教课？",
    item: knowledge,
    onConfirm: () => {
      player.gongxun -= knowledge.cost;
      player.unlocked_knowledge.push(knowledgeId);

      // 打开私教课界面
      openKnowledgeLesson(knowledgeId);

      saveGame();
    }
  });
}
```
