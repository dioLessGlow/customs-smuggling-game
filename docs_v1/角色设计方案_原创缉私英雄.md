# 《缉私十二时辰》原创角色设计方案

> 借鉴真实缉私英雄事迹，但完全原创角色，避免肖像权/名誉权风险

---

## 一、设计原则

### 为什么不要真实人物？

| 风险 | 说明 |
|------|------|
| **肖像权** | 使用真实姓名和形象需授权 |
| **名誉权** | 游戏内角色行为可能被关联到真实人物 |
| **比赛合规** | 部分比赛要求作品完全原创 |
| **改编自由度** | 虚构角色可以加入更多戏剧元素 |

### 借鉴 vs 直接使用

| 方式 | 示例 | 风险 |
|------|------|------|
| ❌ 直接使用 | "郭惠绮，深圳罗湖海关关员" | 高 |
| ✅ 借鉴特质 | "一位50+的女关员，眼神锐利，年查私额千万级" | 低 |
| ✅ 融合多原型 | 把3位英雄的特点融合成1个角色 | 低 |
| ✅ 艺术夸张 | 基于真实事迹但加入虚构情节 | 低 |

## 二、主角（玩家角色）

### 2.1 基础设定

```
【代号】由玩家自定义（默认"见习缉私员-037"）
【性别】可选
【年龄】25岁左右（玩家代入感最强）
【背景】海关学院刚毕业，分配到缉私一线

初始状态：
├─ 职级：见习缉私员
├─ 功勋：0
├─ 警觉：10
├─ 装备：基础扫描仪
└─ 导师：暂无

成长路径：
见习缉私员 → 初级缉私员 → 中级缉私员 → 高级缉私员 → 缉私专家 → 国门卫士
```

- 某南方口岸资深女关员（50+，年查私额千万级）

**融合后角色**：

```
┌─────────────────────────┐
│  【林锐 · 水客识别专家】  │
│                         │
│         👩‍💼            │
│        /████\           │
│       /██████\          │
│         ██              │
│        /  \             │
│       /    \            │
│                         │
│  年龄：50+              │
│  职级：高级缉私员        │
│  特征：短发、眼神锐利、   │
│        走路带风         │
│  习惯：观察旅客先先看脚  │
│        （鞋子磨损判断    │
│         是否频繁出入境）  │
│  口头禅："先看鞋，再看脸，│
│          最后看行李"     │
│                         │
└─────────────────────────┘
```

**背景故事**（借鉴+虚构）：
> 林锐在口岸工作了28年，经手的旅客超过千万人次。她独创的"三步观察法"（看鞋-看脸-看行李）让无数水客无所遁形。有一次，一名旅客穿着崭新的名牌鞋，但鞋底沾着口岸特有的红土——她立刻判断此人刚入境不久，果然查获了未申报的高档手表。

**教学功能**：
- 水客"蚂蚁搬家"手法
- "三步观察法"教学
- 真实案例改编（不提及具体地名和人名）

**引导对话**：
```
林锐："年轻人，水客走私看似简单，实则暗藏玄机。
        他们往往会先放几件合法物品消耗你的注意力，
        从第3件开始，才是真正的考验。"

        "我教你一招——先看鞋。鞋子磨损程度能告诉你
        这人是不是经常出入境。新鞋+旧行李箱=高风险。"

        [播放案例：某旅客新鞋旧箱查获未申报手表]
```

## 五、角色与游戏机制结合

### 5.1 关卡流程

```
【关卡开始】
    ↓
导师出场动画 → 介绍走私类型 + 改编案例
    ↓
玩家操作 → 扫描行李 → 判断
    ↓
判断正确 → 导师鼓励 + 知识补充
    ↓
判断错误 → 导师纠正 + 详细讲解
    ↓
【关卡结束】
    ↓
导师总结 → 隐藏规则揭示 + 功勋奖励
    ↓
解锁图鉴 → 档案员补充历史案例
```

### 5.2 情感曲线设计

```
第1关（水客）：紧张 → 林锐鼓励 → 信心建立
第2关（冻品）：平稳 → 白薇分享防疫故事 → 责任感
第3关（毒品）：压力增大 → 陈锋分享缉毒经历 → 敬畏
第4关（海上）：危险感 → 赵海分享海上缉私 → 勇气
第5关（综合）：挑战 → 老周分享守岛精神 → 坚守
第6关（终极）：高潮 → 小辉纪念 → 情感升华
```

## 六、视觉设计建议

## 八、角色成长系统

### 8.1 玩家角色成长

```
见习缉私员 → 初级缉私员 → 中级缉私员 → 高级缉私员 → 缉私专家 → 国门卫士

晋升条件：
├─ 初级：通关第1周期
├─ 中级：累计功勋100
├─ 高级：通关第3周期
├─ 专家：解锁全部6种走私类型图鉴
└─ 国门卫士：通关第5周期 + 分享10次

职级变化影响：
├─ 制服变化（肩章、胸牌颜色）
├─ 初始警觉增加
├─ 解锁新装备
└─ 导师对话变化（更尊重）
```

### 8.2 导师关系度

```
与每位导师互动 → 增加关系度
关系度等级：
├─ 陌生 → 熟悉 → 信任 → 敬佩 → 传承
└─ 高关系度解锁：导师专属故事、隐藏案例、特殊装备
```
